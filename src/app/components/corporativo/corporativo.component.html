¿Cómo funciona?

La versatilidad del sistema permite diversas opciones
para la distribución del código (Identificación de empleado
o visitantes, código de invitación
para proveedores vía e-mail).

Con un lector fijo o con un dispositivo móvil (Android o iOS),
se lee el código y se permite el acceso a la persona.

Cada código de invitación es único e irrepetible, válido para
una sola entrada y salida, dentro del horario
marcado por el anfitrión.
