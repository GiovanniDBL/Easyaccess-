Registra la informacion de tus visitantes de una manera fácil

Easyaccess es una app para el control de acceso de manera sencilla, podrás tener los registros de las visitas realizadas a tu residencial o empresa mediante la captura de información del visitante y la captura de imágenes los cuales generaran un reporte pdf con el resumen de la visita.

FUNCIONES

validación de horario

La aplicación verifica que el usuario cumpla con las reglas de horario establecidas para su ingreso o salida.

validación de status de usuario

Se valida que el usuario que intenta acceder este activo en el sistema.

Anti-passback

Esta función permitirá el acceso a tus usuarios solo si se ha registrado una salida previamente, lo cual es de gran utilidad para impedir varios ingresos con el mismo acceso.
